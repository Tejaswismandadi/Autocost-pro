About this platform
Built on the should cost methodology used in Tier 1 automotive sourcing
🎯
First-principles cost engineering
Every line item calculated from raw inputs — material weight × commodity price ÷ utilization factor. Machine rate × cycle time ÷ OEE. No black boxes.
📊
Live market data sources
PP resin via ICIS. Steel via Fastmarkets. Zinc and copper via LME. Mexico labor via IMSS/CONASAMI. US labor via BLS OEWS. Freight via DAT.
⚖️
Commercial framework
RFQ normalization for Incoterms and payment terms. LTA APR tracking. Material index passthrough calculation. ECN cost impact quantification.
🔍
Negotiation intelligence
Gap vs should cost by bucket. Sensitivity analysis showing which inputs move cost most. BATNA calculation. Walkaway point. 5-step negotiation script.
